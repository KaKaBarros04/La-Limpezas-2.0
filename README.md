# LA Limpezas 2.0

Landing page moderna e responsiva para **LA Limpezas 2.0**, uma empresa especializada em serviços de limpeza residencial, empresarial, pós-obra e de estofados.

## Funcionalidades

- Hero chamativo com CTA para agendamento
- Seções de serviços detalhadas
- Passo a passo de como funciona
- Depoimentos de clientes
- Botões integrados com WhatsApp para contato direto
- Footer moderno com links e redes sociais
- Design responsivo, moderno e otimizado para experiência do usuário

## Tecnologias

- HTML5
- CSS3 personalizado
- Opcional: TailwindCSS ou Bootstrap para estilização
